

# نشر منصة Sity Experts

## ملخص المنصة

**Sity Experts** هي منصة خدمات مُدارة (Managed Services) باللغة العربية، تربط العملاء بفريلانسرز معتمدين عبر نظام كريديت. المنصة تتضمن:

- **3 أنواع مستخدمين**: عميل (client)، فريلانسر (freelancer)، مدير (admin/team_leader)
- **نظام طلبات**: العميل يرسل طلب → الأدمن يوافق ويعيّن فريلانسر → تسليم → مراجعة جودة
- **نظام كريديت**: باقات اشتراك + كريديت إضافي، خصم كريديت عند إنشاء طلب
- **دفع Kashier**: بوابة دفع إلكتروني مصرية
- **إشعارات Telegram**: بوت تليجرام لإشعارات فورية
- **CMS ديناميكي**: صفحات قابلة للتعديل + Page Builder
- **نظام تدريب**: مسارات تعليمية وكورسات للفريلانسرز
- **بورتفوليو عام**: صفحات عرض أعمال للفريلانسرز
- **دعم فني**: محادثات دعم مباشرة
- **محفظة مالية**: للفريلانسرز مع سحب أرباح

## خطة النشر

### الخطوة الوحيدة: نشر المشروع

المشروع جاهز للنشر. سأقوم بالآتي:

1. **نشر المشروع** عبر أداة Publish بحيث يكون متاح للعامة
2. الرابط سيكون متاح على `*.lovable.app`

### ملاحظات مهمة

- **Edge Functions**: ستُنشر تلقائياً مع أي تعديل
- **قاعدة البيانات**: متصلة بالفعل بمشروع Supabase الخارجي (`looszeychdnjutraogpv`)
- **الـ Frontend**: يحتاج نقرة "Publish" لنشره

### بعد النشر

بمجرد النشر، يمكنك:
- تسجيل حساب عميل جديد واختبار كل الـ flows
- طلب أي تعديلات أو إضافات من ناحية العميل
